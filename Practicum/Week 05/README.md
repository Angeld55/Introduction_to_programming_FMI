**Задача 3:** Да се напишат *функции*, които приемат масив от цели числа и връщат минималното и максималното число в масива.

**Пример:**

Вход:
```c++
[3, 1, 2, 8, 13, 5, 1, 6, 25]
```
Изход:
```c++
std::cout << findMinInArr(arr, size) << endl; //1
std::cout << findMaxInArr(arr, size) << endl; //25
```

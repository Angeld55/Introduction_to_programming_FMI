**Задача 9:**

Да се напише функция, която приема масив, неговия размер и положително число n. Функцията да завърта масива n-пъти надясно.

**Пример:**
Вход:
```c++
[1,2,3,4,5] n = 2
```
Изход:
```c++
[4,5,1,2,3]
```

*Обяснение:* 

* Отместване с 1: Последният елемент се премества в началото, а останалите се местят с една позиция надясно.
    
    Резултат след първото отместване:
    ```c++
     [5, 1, 2, 3, 4]
     ```

* Отместване с 2: Второто отместване се прилага върху резултата от първото.

    Резултат след второто отместване:
    ```c++
     [4, 5, 1, 2, 3]
     ```

**Задача 10:**

Напишете функция `transformNumbers`, която приема масив от цели числа и променя всеки от елементите му, според следните правила:

* Ако числото е *положително* и *четно*, умножете го по броя на положителните числа в масива.

* Ако числото е *положително* и *нечетно*, увеличете го с *най-малкия отрицателен елемент* в масива (ако няма такъв - не променяйте).

* Ако числото е *отрицателно*, сменете знака му и го разделете целочислено на 2.

* Ако числото е *0*, заменете го със средното аритментично на всички елементи в масива (закръглено до цяло число).

*  **Фунцкията трябва да модифицира масива директно, без да се създават нови масиви!**

* **Да се избегнат повторения на изчисления.**

**Пример:**
Вход:
```c++
[2,-3,0,5,-4,6]
```
Изход:
```c++
[6,1,1,1,2,18]
```

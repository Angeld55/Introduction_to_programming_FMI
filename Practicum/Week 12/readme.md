### Пример:
    Вход: 5
    Изход: 1 1 2 3 5

**Задача 2.**

**Задача 3.**

### Пример:
    Вход : [0, 3, 1, 3, 8, 2, 1] 6 
    Процес: [0, 3, 1, 3, 8, 2, {1}] 6 
            [0, 3, 1, 3, 8, {2}, 1] 5 
            [0, 3, 1, {3}, 8, 2, 1] 4 
            [{0}, 3, 1, 3, 8, 2, 1] 0 
    Изход: True!
